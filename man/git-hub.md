git-hub(1) -- Open the Github page for the current repository
=============================================================

## SYNOPSIS

`git-hub`

## DESCRIPTION

If the current repository is not a Github hosted project, `git-hub` will do nothing.

## OPTIONS

  None

## EXAMPLES

  $ git hub
